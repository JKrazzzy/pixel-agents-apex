# Pixel Agents Apex

An independent recreation of `https://github.com/pablodelucca/pixel-agents` for OpenClaw agent implementation tracking. This repository aims to replicate the architecture and behavior of the original project.

## Run Instructions

To set up and run this project:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/JKrazzzy/pixel-agents-apex.git
    cd pixel-agents-apex
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Build the project:**
    ```bash
    npm run build
    ```
    This command will compile the main extension and build the webview UI components.
